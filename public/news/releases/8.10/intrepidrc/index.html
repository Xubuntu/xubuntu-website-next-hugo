Xubuntu 8.10 Release Candidate now available

  • October 24, 2008
  • 8.10

The Release Candidate of Xubuntu 8.10 “Intrepid Ibex” has arrived. This is a testing build designed for developers and is not suggested for anyone looking for a stable system. If you would like to see what is new in the beta release, please see the Release Notes.

Visit the download page to give it a try. Be sure to report your testing results to help the Xubuntu developers - see the ISO testing tracker, Bugs can be reported in Launchpad.

Before testing, be sure to take note of the Known Issues.

If you are looking for support), please see the #ubuntu+1 or #xubuntu IRC channels on irc.freenode.net.
